# Architecture

## Tech Stack

- **Frontend:** Next.js (SSR), deployed via SST + OpenNext
- **Backend:** AWS Lambda, API Gateway
- **Database:** DynamoDB
- **Auth:** AWS Cognito (Google / Facebook OAuth)
- **CDN:** AWS CloudFront
- **Rate Limiting:** Upstash Redis
- **IaC:** AWS CDK + SST
- **Observability:** AWS CloudWatch
